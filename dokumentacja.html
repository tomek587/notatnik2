Dokumentacja Notatnika - Aplikacja Python

Wprowadzenie

Notatnik to aplikacja oparta na Pythonie, która umożliwia użytkownikowi tworzenie, edytowanie, przeglądanie oraz usuwanie notatek. Program wymaga bazy danych MySQL.

Program pozwala na:

  • Rejestrację i logowanie użytkowników
  • Tworzenie, edytowanie oraz usuwanie notatek
  • Przeszukiwanie notatek za pomocą wyszukiwarki
  • Bezpieczne przechowywanie danych użytkownika i jego notatek w bazie danych

Wymagania

Aby uruchomić aplikację, należy posiadać następujące biblioteki:

  • Python w wersji 3.6 lub nowszej
  • tkinter - do tworzenia interfejsu
  • Pillow - do obsługi obrazów
  • mysql-connector - do połączenia z bazą danych
  • mysql - do połączenia z bazą danych
  • configparser - do przechowywania danych bazy danych w pliku config.ini

Aby zainstalować te biblioteki, uruchom następujące polecenie:

        pip install tkinter Pillow mysql-connector configparser mysql

Uruchomienie Aplikacji

Kroki, aby uruchomić program:

  1. (Jeśli potrzebne) Skonfiguruj nazwę hosta, użytkownika i hasło w pliku config.ini
  2. Uruchom plik main.py, np. z poziomu terminala: python main.py, aby uruchomić aplikację

Jak Działa Program?

1. Logowanie/Rejestracja

Po uruchomieniu aplikacji użytkownik jest proszony o podanie loginu i hasła. Jeśli użytkownik istnieje w bazie danych, może się zalogować. Jeśli konto nie istnieje, program utworzy nowe konto i zaloguje użytkownika.

2. Notatnik

Po zalogowaniu użytkownik zostaje przeniesiony do głównej aplikacji, w której może przeglądać swoje notatki. Użytkownik może:

  • Przeglądać notatki - notatki wyświetlane są na liście z krótkim podglądem oraz datą ich utworzenia
  • Dodawać notatki
  • Edytować istniejące notatki
  • Usuwać notatki
  • Wyszukiwać notatki

3. Wylogowanie

Użytkownik może się wylogować. Po wylogowaniu aplikacja wraca do ekranu logowania.

Struktura Bazy Danych

Program korzysta z bazy danych MySQL, która zawiera dwie tabele:

  • users - przechowuje dane użytkowników (login i zaszyfrowane hasło)
  • notatki - przechowuje notatki, które są powiązane z użytkownikami za pomocą klucza (user_id)

Baza danych jest tworzona automatycznie przy pierwszym uruchomieniu programu, o ile konfiguracja bazy danych jest poprawna.
